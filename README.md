# Hybrid Quantum-Classical Toxicity Prediction

A Quantum Machine Learning (QML) project that predicts molecular toxicity using a hybrid Quantum-Classical framework. The system combines quantum feature extraction through Qiskit with classical machine learning techniques to classify chemical compounds as toxic or non-toxic.

---

## Project Overview

Toxicity prediction plays a crucial role in drug discovery, chemical safety assessment, and pharmaceutical research. Traditional toxicity testing methods are often expensive and time-consuming.

This project leverages Quantum Machine Learning to explore how quantum circuits can enhance feature representation and improve predictive modeling for molecular toxicity analysis.

---

## Objectives

* Analyze molecular compound data for toxicity prediction
* Extract meaningful chemical features from molecular structures
* Implement quantum feature encoding using Qiskit
* Develop a hybrid Quantum-Classical machine learning pipeline
* Evaluate classification performance using standard metrics
