# qml-toxicity-prediction
Hybrid Quantum-Classical Toxicity Prediction using Qiskit and Machine Learning
## Overview
This project implements a Hybrid Quantum-Classical Machine Learning framework for molecular toxicity prediction.

## Technologies Used
- Python
- Qiskit
- RDKit
- Scikit-Learn
- NumPy
- Pandas

## Features
- Molecular fingerprint generation
- Quantum feature extraction
- Toxicity classification
- Data visualization and analysis

## Results
Achieved 97.5% classification accuracy using a Random Forest classifier integrated with quantum-generated features.
